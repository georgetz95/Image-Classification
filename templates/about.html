This image classification model was created using 2,057 images belonging in one of the following classes:

['Bear', 'Cat', 'Chicken', 'Cow', 'Deer', 'Dog', 'Duck', 'Eagle', 'Elephant', 'Human', 'Lion', 'Monkey', 'Mouse', 'Natural', 'Panda', 'Pigeon', 'Rabbit', 'Sheep', 'Tiger', 'Wolf']

Each image's features were extracted using Histogram of Oriented Gradients (HOG), which is a feature descriptor often used with image data. The complete image is broken down into smaller regions and for each region, the gradients and orientation are calculated. It focuses on the structure or shape of each portion of an image and extracts the edge features and their direction. It then generates a histogram for each one of these regions using the gradients and orientations of the pixel values.

The classification model is a Stochastic Gradient Descent Classifier that was hyperparameterized using GridSearchCV.
